# open-webrtc

[jellybeans511/open-webrtc: Don't abuse...](https://github.com/jellybeans511/open-webrtc)から生まれた WebRTC で遠隔監視 Web アプリ。Don't abuse too。

## リンク

- [https://kwmk.github.io/open-webrtc/](https://kwmk.github.io/open-webrtc/)

## 主な機能

- リアルタイム配信
- カメラ選択
- 1対多配信
- 実解像度・FPS・コーデック確認
- カメラ切り替え
- ideal height指定
- できるだけ自動再接続
